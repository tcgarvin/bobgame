# Bob's World Viewer

Phaser 3 visualization for the world simulation.

## Quick Start

```bash
npm install   # first time only
npm run dev   # start dev server at http://localhost:5173
```

## Scripts

- `npm run dev` - Start development server with hot reload
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Stack

- **Vite** - Build tool
- **TypeScript** - Type safety
- **Phaser 3** - Game framework

## Status

**Milestone 1 complete**: Static map rendering with DawnLike tileset, entity sprites, camera controls (WASD/arrows to pan, scroll to zoom).

**Milestone 4 pending**: Live connection to world server via gRPC-Web or WebSocket bridge.
